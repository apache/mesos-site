163 // The `UriVolumeProfileAdaptor` is an example VolumeProfile module that
164 // takes a URI as a module parameter and fetches that URI periodically.
165 // The fetched data is parsed into the required CSI protobufs
166 // (which also acts as validation).
167 //
168 // If there is an error during fetching, any previously fetched results
169 // will be used until fetching is successful.
170 //
171 // This module does not filter return results based on `CSIPluginInfo::type`
172 // and assumes that all fetched profiles are meant for all resource providers.
173 //
174 // See `Flags` above for more information.

186  virtual process::Future<hashset<std::string>> watch(
187  const hashset<std::string>& knownProfiles,
188  const std::string& csiPluginInfoType) override;
