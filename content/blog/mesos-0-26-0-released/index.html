Apache Mesos 0.26.0 Released

The latest Mesos release, 0.26.0, is now available for download. 103+ bugfixes and improvements made it into this release. For full release notes with all features and bug fixes, please refer to the CHANGELOG.

Upgrades

Rolling upgrades from a Mesos 0.25.0 cluster to Mesos 0.26.0 are straightforward. There are just some minor, backwards compatible deprecations. Please refer to the upgrade guide for detailed information on upgrading to Mesos 0.26.

Try it out

We encourage you to try out this release and let us know what you think. If you run into any issues, please let us know on the user mailing list and IRC.
