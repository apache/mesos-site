Apache Mesos 0.26.1 and More Released

The latest Mesos 0.26.1, 0.25.1 and 0.24.2 are now available for download. The applicable subset of the following issues were backported.

  • MESOS-920 - GLOG performance.
  • MESOS-3046 - UUID performance.
  • MESOS-3051 - Port Ranges performance.
  • MESOS-3052 - Allocator filter performance.
  • MESOS-3307 - Configurable task/framework history.
  • MESOS-3560 - JSON-based credential files.
  • MESOS-3602 - HDFS.
  • MESOS-3738 - Mesos health check within docker container.
  • MESOS-3773, MESOS-4069 - SSL.
  • MESOS-3834 - Agent upgrade compatibility
  • MESOS-4106 - Health checks.
  • MESOS-4235 - /state endpoint performance.
  • MESOS-4302 - Offer filter timeout fix for backlogged allocator.
  • MESOS-4687 - Fixed point resources math.
  • MESOS-4711 - Libevent.
  • MESOS-4979 - Deletion of special files.
  • MESOS-5021 - Memory leak in subprocess.

Full release notes are available in the release CHANGELOG.

Upgrading to 0.26.1 can be done seamlessly on a 0.26.0 cluster. If upgrading from an earlier version, please refer to the upgrades documentation.
