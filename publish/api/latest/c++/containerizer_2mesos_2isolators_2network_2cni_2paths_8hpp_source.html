29 // The root directory where we keep the information of CNI networks that each
30 // container joins. The layout is as follows:
31 // /var/run/mesos/isolators/network/cni/
32 // |- <ID of Container1>/
33 // | |-- ns -> /proc/<pid>/ns/net (bind mount)
34 // | |-- <Name of CNI network 1>/
35 // | | |-- network.conf (JSON file to keep the CNI network configuration)
36 // | | |-- ifname1/
37 // | | |-- network.info (JSON file to keep the output of CNI plugin)
38 // | |-- <Name of CNI network 2>/
39 // | | |-- network.conf
40 // | |-- ifname2/
41 // | |-- network.info
42 // |-- <ID of ContainerID 2>/
43 // | ...
